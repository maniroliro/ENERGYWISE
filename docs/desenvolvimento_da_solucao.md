# Desenvolvimento da Solução – EnergyWise

Este documento descreve de forma detalhada a lógica, critérios e justificativas da
solução EnergyWise, criada para promover eficiência energética no ambiente de trabalho.

## 1. Estratégia Utilizada
A solução foi construída com base em:
- dados de consumo horário,
- identificação de padrões críticos,
- detecção de picos acima de 80 kWh,
- análise estatística simples (mínimo, máximo, média).

## 2. Critérios de Otimização
A solução considera consumo acima de 80 kWh como faixa crítica, indicando:
- sobrecarga de equipamentos,
- climatização excessiva,
- iluminação desbalanceada.

## 3. Regras Implementadas
- Identificar todos os horários onde o consumo excede o limite.
- Calcular a energia desperdiçada.
- Estimar economia mensal e anual.
- Registrar recomendações no relatório final.

## 4. Recomendações Propostas
Com base na análise:
- Ajustar climatização entre 10h e 13h.
- Usar sensores de presença para iluminação.
- Redistribuir carga de equipamentos.
- Considerar uso de energia solar nesses horários.

## 5. Contribuição para o Futuro do Trabalho
A solução fornece:
- tomada de decisão baseada em dados,
- eficiência operacional,
- redução de custos,
- menor impacto ambiental,
- base para automação inteligente.
